## 🎯 Agent Assignment Algorithm

```python
def assign_agents_to_repo(repo):
    """
    Dynamically assign agents based on repo characteristics
    """
    base_agents = 50  # Minimum per repo

    # Factor 1: Tier
    tier_multiplier = {
        'tier-1': 10,
        'tier-2': 5,
        'tier-3': 2,
        'tier-4': 1
    }

    # Factor 2: Activity
    activity_score = repo.commits_per_day * 10

    # Factor 3: Complexity
    complexity_score = repo.lines_of_code / 1000

    # Factor 4: Dependencies
    dependency_score = len(repo.dependencies) * 5

    total_agents = (
        base_agents +
        (tier_multiplier[repo.tier] * 100) +
        activity_score +
        complexity_score +
        dependency_score
    )

    return int(total_agents)
```

### Example Assignments:
- **blackroad-os-core:** Tier 1, high activity → 2,000 agents
- **blackroad-os-docs:** Tier 2, medium activity → 500 agents
- **lucidia-earth:** Tier 3, low activity → 100 agents
- **experimental-quantum:** Tier 4, low activity → 50 agents
